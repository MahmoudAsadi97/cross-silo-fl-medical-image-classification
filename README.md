# Cross-Silo Federated Learning for Medical Image Classification

Bachelor project on cross-silo federated learning for privacy-preserving medical image classification.

## Week 1 Goals
- Finalize scope
- Set up project structure
- Verify environment and GPU
- Prepare configs and documentation skeleton

## Main Structure
- `src/` core source code
- `configs/` experiment and module configs
- `scripts/` runnable entry points
- `notebooks/` analysis notebooks
- `results/` generated outputs
- `docs/` manuals and report material
- `planning/` weekly planning and notes
